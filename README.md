## Installation ğŸ› ï¸
  **Step 1**: Fork the repository. You can fork the repository by clicking on the fork button on the right hand side below the profile.<br> 
  
  **Step 2**: Clone your forked repository. Replace yourusername with your GitHub Username. 
  
  ```sh
git clone "https://github.com/yourusername/Metaverse-Messenger.git"
``` 
  **Step 3**: After cloning the repository. Go to the repository. 
  
  ```sh
cd Metaverse-Messenger
``` 
  **Step 4**: Run npm init -y. It will install all the packages and dependencies used in the project. 
  
  ```sh
npm init -y 
```
  **Step 5**: Run npm install next. This will install Next.js in your local repository.  
  
  ```sh
npm install next 
``` 
  **Step 6**: Run npm run dev. This will start the project in your local machine ğŸ–¥ï¸. 
  
  ```sh

npm run dev 
``` 
  **Step 7**: To run the project in your local machine, you need to go to your local server which is running on port 3000. Go to the browser and paste the link given below ğŸ‘‡: 
  
  ```sh
 http://localhost:3000/
```
